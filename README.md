# Deface-Webdav
This webdav web defce method is quite popular, previously defacement usually used Linux devices such as Kali Linux, and now you can practice this method on Android using the Termux helper application.

how to install
$ sudo apt -y update && sudo apt -y install wget openssl curl

$ pip2 install --upgrade pip

$ pip2 install urllib3 

$ pip2 install chardet certifi idna requests

$ git clone https://github.com/spyschools/Deface-Webdav.git

$ cd Deface-Webdav

run
$ python2 webdav.py exampel.com index.html

index.html: is a deface script, please change it according to your wishes as long as the script name is: index.html
